# 表白微信小程序

#### 介绍
基于uni-app、uView UI开发的用于表白的微信小程序项目

#### 软件架构
uni-app、uView UI


#### 安装教程

此项目不需要安装

#### 使用说明

1.  下载Hbuilder
2.  下载微信开发者工具
3.  导入本项目到Hbuilder
4.  Hbuilder点击运行-运行到小程序模拟器-微信开发者工具
